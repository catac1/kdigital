# MQTT broker

1. pull `eclipse/mosquitto` from docker hub
2. create mosquitto folder under `c:\Users\kst` and `mosquitto.conf` file inside of the folder.
```conf
# 기본 MQTT (TCP) : ios, android, iot
listener 1883
protocol MQTT

# WebSocket 지원: web, react 연동
listener 9001
protocol websockets

# 인증 (테스트용)
allow_anonymous true
```
3. create docker container via cli: because port is not exposed on eclipse/mosquitto image
```cmd
docker run -d --name mosquitto -p 1883:1883 -p 9001:9001 ^
-v C:/Users/kst/mosquitto/mosquitto.conf:/mosquitto/config/mosquitto.conf ^
eclipse/mosquitto:1.4.8
```


# Recap

## createSlice [Reference Documentation](https://redux-toolkit.js.org/api/createSlice)

### Definition of reducer
```js
    import { createSlice } from "@reduxjs/toolkit";

    const <slice name> = createSlice({
        name: <slice name: string>,
        initialState: <value | function>,
        reducers:
            [<action>(state, [action]) => {
                state.<property>
                action.payload.<property>
            }, ...]
    });

    export const { [<action> ...] } = <slice>.actions;
    export default <slice>.reducer;


```

### Access the state
```js
    import { useSelector } from 'react-redux';

    const { [<property of state>, ...]  } = useSelector((state) => state.<slice name>);
```

### Action on the state

```js
    import { [<action of reducer>, ...] } from '../path/to/sliceFilename.jsx';
    import { useDispatch } from 'react-redux';

    const dispatch = useDispatch();
    dispatch(<action of reducer>({
        [<action payload property name>: < action payload property value>, ...]
    }));

```


### Define store

```js
    import { configureStore } from "@reduxjs/toolkit";
    import {<imported slice name but name is as Reducer>} from "./path/to/sliceFile";

    export const store = configureStore({
        reducer: {
            [<reducer name>: <imported slice name as reducer>, ...]
        }
    });
```

### Use store

```js
    import { Provider } from 'react-redux'
    import { store } from './path/to/storeFile'

    createRoot(document.getElementById('root')).render(
        <BrowserRouter>
            <Provider store={store}>
                <App />
            </Provider>
        </BrowserRouter>,
    );
```
