## Overview

This commit added item detail viewing and image management features:
- A new **Item Detail** page (`/item/detail`)
- A new **Item Image Insert** page (`/itemimage/insert`)
- Row click navigation from the item list to the detail page
- New routes wired in `App.jsx`

## Files Changed (4 files, +257 / -2)

### 1. `src/App.jsx` (+4 lines)
Added imports and routes for the two new pages (`ItemDetail`, `ItemImageInsert`).

### 2. `src/pages/Item.jsx` (+11, -2 lines)
Added `useNavigate` and an `onRow` click handler so clicking any row navigates to `/item/detail?code=XXX`.

### 3. `src/pages/ItemDetail.jsx` (new, +174 lines)

Displays full details for a single item and manages its images.

**State:**
| State       | Purpose                                   |
| ----------- | ----------------------------------------- |
| `row`       | Item detail data (name, price, qty, etc.) |
| `imageList` | Images associated with the item           |
| `chk`       | Toggle to re-fetch after delete/update    |
| `modalOpen` | Controls the image-change modal           |
| `fileList`  | Files staged in Dragger for upload        |
| `changeNo`  | Image ID selected for update              |

**API calls:**
| Endpoint                       | Method | Purpose                      |
| ------------------------------ | ------ | ---------------------------- |
| `/api/item/select.do?code=`    | GET    | Fetch item details           |
| `/api/itemimage/list.do?code=` | GET    | Fetch image list             |
| `/api/itemimage/delete.do`     | DELETE | Delete an image              |
| `/api/itemimage/update.do`     | PUT    | Replace an image (multipart) |

**UI:** Item field display, inline image gallery with Delete/Change buttons per image, modal with Ant Design Dragger for single image replacement.

### 4. `src/pages/ItemImageInsert.jsx` (new, +70 lines)

Uploads multiple images for an item.

**API calls:**
| Endpoint                           | Method | Purpose                             |
| ---------------------------------- | ------ | ----------------------------------- |
| `/api/itemimage/multipleinsert.do` | POST   | Upload multiple images via FormData |

**UI:** Ant Design Dragger (up to 10 files), navigation buttons back to detail/list, submit button.
