## MongoDB Collection
```
collection is analogous to table in relational db
```

### Selecting collection
```
db.get_collection(<collection name>)
```

### getting to dict from collection
```
collection.find( <query>, <projection>)
```
- `{}` query is getting all items.
- `{"column name": 0 or 1}` is include or exclude from the projection

### convert to dataframe
- Must convert list from dict to create DataFrame
```
pd.DataFrame(list(<dict name>))
```

### Pandas DataFrame join
```
pd.merge(<left_df_name>, <right_df_name>, [on=<join column name>| left_on=<jcn> | right_on=<jcn>] )
```

### Pandas DataFrame projection
```
<df name>[[<column name>,...]]
```
- ex
```
foo[['a', 'b', 'c']]
```
