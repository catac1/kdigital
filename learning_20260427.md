# NOTE for DATABASE design principle

## 2nf requirement
    - 모든 컬럼은 주 키 전체에 종속 되어야 함. 일부 주 키에 종속되는 컬럼이 있으면 위반.
    - 
## Problems when 2nf is violated.
    - Insert Anomaly: 쓸데 없는 정보가 들어가야함
    - Delete Anomaly: 정보가 유실 될 수 있음
    - Update Anomaly: 일부만 수정되어 데이터의 일관성 깨짐
