#  DML 수정 실습

- UPDATE <table name> SET <column name>=<new value>[, <column name>=<new value>...] 
  WHERE <column name>=<matching value>;

- DELETE FROM <table name> WHERE <column name>=<matching value>;

- SELECT * FROM <table name>;
- SELECT <table name>.<column name>[, <table name>.<column name>] FROM <table name>;
- SELECT <table name>.* FROM <table name>;
- SELECT <alias name>.<column name>[, <alias name>.<column name>] FROM <table name> <alias name>;
- SELECT <alias name>.* FROM <table name> <alias name>;

- INNER JOIN
- OUTER JOIN: 빈칸이 Null로 채워짐
    - RIGHT JOIN, LEFT JOIN
    - FULL JOIN
  
- CREATE VIEW <= this is DDL

# 회원정보 뷰 생성

- 매번 Join 없이 회원정보를 가져오기위해 View를 생성함 
- View는 Table처럼 사용 가능 함
- View는 읽기 전용
  
  ```sql
   CREATE VIEW <view name> AS 
  ( 
    SELECT <columns from both tables> FROM <table name a> a
    INNER JOIN <table name b> b
    ON a.<column name> = b.<column name>;
  );
  ```
