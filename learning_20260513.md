## Manipulating data with pandas

### rename
```python
x = {[<old column name>:<new column name>,...]}
<df variable>.rename(columns=x)
```
기본옵션은 `inplace=False`라 실제로 데이터를 바꾸지 않음
따라서, 새 변수에 기록하거나 `inplace=True` 로 실행 해야함
  
### to_datetime
```python
pd.to_datetime(<variable>[<column name>], <datetime format string>)
```

### 데이터 저장 불러오기
#### Option1
```python
<dataframe name>.to_pickle(<path>)
pd.read_pickle(<path>)
```
#### Option2
- save
```python
import shelve

with shelve.open('conda_store') as db:
    db['my_saved_data'] = my_variable  # Saves instantly to a local file
```
- load
```python
import shelve

with shelve.open('conda_store') as db:
    my_variable = db['my_saved_data']
```

### When to use which in Jupyter?
Use pickle if you just need to "freeze" a single specific variable to disk 
so you can pick up where you left off later.
Use shelve if you are running an experiment and want to save many different variables 
(e.g., features, labels, model_params) and only want to pull one out at a time without
reloading everything.

### injecting token header
```js
api.interceptors.request.use(
    (config) => {
        // 토큰을 redux store에서 가져와서 config.header.Authorization field에 설정
        const { token } = store.getState().logged;
        if ( token ) {
            config.headers.Authorization = `Bearer ${token}`;
        }
        return config;
    },
    (error) => {
        return Promise.reject(error);
    }
)

```
